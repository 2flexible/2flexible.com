# Installation
You can download core package from npm or you can get minified version from github packages.
## NPM
```shell
npm install 2flex
```

If you want to add more blocks tou your existing project just use 2flex npx command and name of the package you want to use.
[List of the blocks]()
```shell
npm 2flex add [block name] 
```
